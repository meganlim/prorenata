Instructions
Within each organ system, there is a classification map and maps for each disease state. The classification map provides relationships over all drug classes in that block while the disease state maps are for only that indication.

To reveal information, hover mouse over drug class. Upon clicking, a text box will appear with pertinent information. To close, click anywhere on the pop up textbox. Note: you will need to close an opened box before clicking another drug class.
